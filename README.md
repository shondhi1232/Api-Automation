# Api-Automation

## Technology and tools used
-Postman
-Newman

## senario
 #As Admin
  -Login in to the system
  -Create Agent
  -Create Customer
  -Deposit to Agent
 
 #As Agent
  -Login in to the system
  -Deposit to Customer
  
 #As Customer
  -Login in to the system
  -Withdraw money
  -Send money
  -Check balance
  
## How to run this project
  -clone this project
  -hit the following command
  ``` npm i```
  ``` npm test ```
  
## Pre requisite
 -Node.js must be installed
